#  Система сегментации изображений на основе гауссовских смесей (GMM)

> Курсовой проект по кластерному анализу и обработке изображений. 
> Десктопное приложение для автоматической сегментации изображений с использованием алгоритма Gaussian Mixture Models (GMM).

# Описание проекта
Приложение предназначено для выделения однородных областей (сегментов) на цифровых изображениях. В основе лежит алгоритм кластеризации пикселей по методу гауссовских смесей (GMM), 
который моделирует распределение цветов/интенсивностей пикселей в виде смеси нескольких многомерных нормальных распределений. Это графический инструмент для создания коллажей,
но так жк разработан для демонстрации понимания принципов ООП и SOLID, а так же базовых знаний кластерного анализа и разработки алгоритмов. Представлено  сравнение моего алгоритма с Accord.Net
атематический аппарат и бизнес-логика полностью изолированы от пользовательского интерфейса.

#  Основные возможности
-  Загрузка изображений различных форматов.
-  Гибкая настройка параметров кастомного алгоритма GMM (количество кластеров, порог сходимости, максимальное число итераций, размер подвыборки для обучения).
-  Выполнение кластеризации пикселей с использованием GMM.
-  Визуализация результата сегментации и наложение маски на исходное изображение.
-  Сохранение обработанных результатов  в файл .png/.jpg.

#  Архитектура
Проект построен по многоуровневой архитектуре:
1. GMMLogics — ядро системы. Содержит математические модели, алгоритмы кластеризации (EM-алгоритм для GMM) и работу с данными изображений. Не зависит от UI.
2. WindowAppearance — слой представления (UI). Отвечает за взаимодействие с пользователем, отрисовку изображений и передачу команд в бизнес-логику.
3. Tests — набор модульных тестов для проверки корректности математических вычислений и граничных случаев.
